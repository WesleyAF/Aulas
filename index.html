Sobre Variáveis


/*
  Variáveis 
  Em JavaScript, variáveis são contêineres para armazenar dados, declaradas com as palavras-chave var,
  let ou const. const é para valores que não mudam, let é para valores que podem ser reatribuídos,
  e var é uma forma mais antiga considerada obsoleta para novos códigos. 

  As variáveis podem conter diversos tipos de dados como números, textos (strings) e booleanos (verdadeiro/falso),
  e são essenciais para a manipulação de dados em programas. 

  Diferenças entre var, let e const:

  const (constante): É usada para declarar variáveis que terão um valor fixo e que não podem ser alteradas após sua declaração. 
  Exemplo: const PI = 3.14; 

  let: É usada para declarar variáveis que podem ter seu valor alterado durante a execução do código. 
  Exemplo: let nome = "Maria"; 

  var: É a forma mais antiga de declarar variáveis, mas é menos utilizada em códigos modernos, sendo let e const recomendadas.
  Ela tem um escopo mais amplo, e seu uso é considerado obsoleto. 
*/

Sobre Estrutura de Condição


/*
  Em JavaScript, estruturas condicionais como if, else e else if
  permitem que o código tome decisões e execute blocos diferentes 
  com base em condições (verdadeiras ou falsas). 

  - if: executa se a condição for verdadeira
  - else: executa se a condição for falsa
  - else if: verifica outras condições em sequência
*/
